# Guess Number 

Классическая игра «Угадай число» на 7 языках программирования. Каждая версия поддерживает:

- **Генерация случайного числа** в заданном диапазоне
- **Выбор сложности** (легкий, средний, сложный) – влияет на диапазон и количество попыток
- **Ограниченное количество попыток** (например, 10 для среднего уровня)
- **Подсказки** – «Слишком высоко» / «Слишком низко»
- **Отслеживание лучшего результата** (минимальное количество попыток за сессию)
- **Повтор игры** без перезапуска программы
- **Ввод с валидацией** (только числа, проверка границ)

## Реализации

| Язык | Технология | Особенности |
|------|------------|-------------|
| Python | Console | Цветной вывод (опционально) |
| Go | Console | Простая консоль |
| Rust | Console | Безопасный ввод |
| C# | Console | Классический .NET |
| JavaScript | Web | Браузерная версия с интерфейсом |
| TypeScript | Web + классы | Типизированная версия |
| Java | Console | Кроссплатформенный |
